FAQ

O que é o FAQ?

O FAQ é uma lista de perguntas e respostas que ajudam a resolver as tuas duvidas sobre o website

O que é o dashboard, IP, porta?

1. O dashboard é a página que aparece quando tu entras no site. É a página que mostra a temperatura.

2. O IP é um conjunto de números que identifica um computador, neste caso, o servidor.

3. A porta é a porta que o site vai abrir para ter acesso ao servidor.

Como posso mudar o meu nome?

Ao tocar em cima do teu nome irá aparecer um pop-up em que tu
poderás escrever o teu novo nome

Porque está a aparecer "Erro" no dashboard?

O servidor está desligado, para mais informações sobre o servidor vai á página "Server"

Como posso estar a par das atualizações do website?

Para estar a par das atualizações do website clica no botão
"Versão" que está no canto inferior direito do dashboard
